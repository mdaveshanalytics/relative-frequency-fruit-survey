# 📊 Relative Frequency Table Project

## 📌 Topic: Favorite Fruit Survey
This project analyzes survey data to calculate **frequency** and **relative frequency**, helping to understand user preferences in a clear and structured way.

## 📂 Dataset

### Raw Data:
•Apple  
•Banana  
•Apple  
•Mango  
•Banana  
•Apple  
•Mango  
•Banana  

## 📊 Frequency Analysis

| Fruit  | Frequency |
|--------|----------|
| Apple  | 3        |
| Banana | 3        |
| Mango  | 2        |

### ✔️ Key Observation:
- Apple and Banana have the highest occurrence.
- Mango appears less frequently in comparison.

## 📈 Relative Frequency Analysis

**Formula:**
Relative Frequency = Frequency / Total Observations  

Total Observations = 8  

| Fruit  | Relative Frequency | Percentage |
|--------|------------------|------------|
| Apple  | 0.375            | 37.5%      |
| Banana | 0.375            | 37.5%      |
| Mango  | 0.25             | 25%        |

## 🔍 Detailed Analysis & Insights

### 1. Popularity Comparison
- Apple and Banana are **equally preferred**, each contributing **37.5%** of total responses.
- Mango accounts for only **25%**, indicating comparatively lower preference.

### 2. Distribution Insight
- The data shows a **balanced distribution** between Apple and Banana.
- No single fruit dominates the dataset → indicates **diverse user preference**.

### 3. Decision-Making Insight
- If this were a real business scenario (e.g., inventory planning):
  - Apple and Banana should be stocked in **equal and higher quantity**.
  - Mango stock can be **slightly lower but still significant**.

### 4. Statistical Understanding
- Relative frequency provides a **proportional view** instead of just raw counts.
- Helps in comparing categories even when dataset size changes.

### 5. Data Quality Note
- Small dataset (n=8), so results are **indicative but not conclusive**.
- Larger sample size would improve accuracy and reliability.

## Project Outcome

- Successfully converted raw data into structured insights  
- Applied basic statistical concepts (frequency & relative frequency)  
- Derived meaningful conclusions from simple dataset
